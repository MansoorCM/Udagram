## Dependencies

- Node v14.15.1 (LTS) or more recent. (v14.17.5 was used for the project)

- npm 6.14.8 (LTS) or more recent.

- AWS CLI v2.

- A RDS database running Postgres.

- Two S3 buckets, one for hosting uploaded pictures and one for the hosting the static website.

- CircleCI for setting up the CI/CD pipeline.
